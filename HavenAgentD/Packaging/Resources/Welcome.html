HAVEN Agent (haven-agentd)

haven-agentd is a small background agent for macOS. It lets a HAVEN entity request actions on this Mac through an explicit, locally-approved policy — it never gives the cloud a free scripting channel.

This installer places three command-line tools and a startup template on the Mac. There is no app icon and no window — that is expected. You drive it from Terminal, and it runs quietly in the background once set up.

After installation, the final screen shows the exact commands to run. Nothing starts or connects to any server until you run setup or consume a separate, single-use correspondence invitation.
