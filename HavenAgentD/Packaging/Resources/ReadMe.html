What this installs

  • /usr/local/libexec/havenagent/haven-agentd — the agent
  • /usr/local/libexec/havenagent/sprout — scaffold join/bootstrap helper
  • /usr/local/libexec/havenagent/haven-correspondence-mcp — signed, messages-only Claude/Codex bridge
  • /usr/local/share/havenagent/ — LaunchAgent template + guides
  • symlinks so haven-agentd and haven-correspondence-mcp work from any Terminal

Your per-user data lives under ~/Library/Application Support/HAVENAgent/ and is created the first time you run setup.

What it does NOT do

  • It does not start automatically after install — you run setup first.
  • It does not connect anywhere until you import provisioning evidence.
  • It cannot run arbitrary scripts from the cloud — only locally allow-listed actions.
  • The correspondence MCP cannot run any local action. Message text never grants execution authority.

Binaries are signed and notarized by Stiftelsen Digipomps.
